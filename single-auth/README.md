# Single Auth

Simple single-app Logto auth demo.

## Setup
1. Install dependencies:
   - npm install
2. Create env file:
   - copy .env.example to .env
3. Start app:
   - npm start

## Environment
Set these values in .env:
- LOGTO_APP_ID
- LOGTO_APP_SECRET
- LOGTO_ENDPOINT
- LOGTO_BASE_URL
- SESSION_SECRET
- PORT

## Notes
- .env is ignored by git.
- Use .env.example as a template.
